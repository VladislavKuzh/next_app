### Prerequisites

**Node version 16.x**

### Cloning the repository

```shell
git clone https://github.com/VladislavKuzh/next_app.git
```

### Install packages

```shell
npm i
```

### Setup .env file

```js
MONGO =

GOOGLE_CLIENT_ID =
GOOGLE_CLIENT_SECRET =

NEXTAUTH_SECRET =
NEXTAUTH_URL =
```

### Start the app

```shell
npm run dev
```
